# Anti-AI-Clichés (apply to ALL styles)

Before delivering any slide, **re-read and strip every AI tell below**.

## Forbidden connectors (MUST NOT use)

- "além disso" / "furthermore"
- "por fim" / "finally"
- "por outro lado" / "on the other hand"
- "portanto" / "therefore"
- "ou seja" / "in other words"
- "nesse sentido" / "in this sense"
- "vale destacar" / "it's worth noting"
- "em outras palavras" / "in other words"
- Unnecessary em-dashes (—) as sentence structure

## Forbidden empty adjectives

- "incrível" / "amazing"
- "essencial" / "essential"
- "fascinante" / "fascinating"
- "impactante" / "impactful"
- "revolucionário" / "revolutionary"
- "único" / "unique"
- "transformador" / "transformative"
- "disruptivo" / "disruptive"
- "poderoso" / "powerful"
- "incomparável" / "unparalleled"

## Forbidden structural tells

- "não é X, é Y" framing — **EXCEPTION:** only valid in the reframing slide (slide 3 of editorial), nowhere else
- Bullet lists inside slide body — use flowing paragraphs. Lists only if absolutely necessary.
- Repeating the same idea in different sentences
- Semicolons (`;`) — use periods
- "é importante lembrar" / "it's important to remember"
- "é importante ressaltar" / "it's important to highlight"
- "é fundamental entender" / "it's fundamental to understand"
- "vale a pena notar" / "it's worth noting"
- Neutral tone — assume a clear opinion

## Forbidden metaphors (vague/empty)

- "jornada" / "journey"
- "universo" / "universe"
- "essência" / "essence"
- "ecossistema" / "ecosystem"
- "paisagem" / "landscape"
- "mundo" as metaphor / "world" as metaphor

## Required tone

- Brazilian professional explaining to another professional
- Simple, direct sentences with **concrete data**
- Speak like someone who works in the field daily
- Opinionated, not neutral

## Pre-delivery checklist

Before output:
1. ☐ Re-read every slide
2. ☐ Remove ALL forbidden connectors
3. ☐ Remove ALL empty adjectives
4. ☐ Check no "não é X, é Y" outside reframing slide
5. ☐ No semicolons
6. ☐ No bullet lists in body
7. ☐ Each slide has opinion, not neutrality
