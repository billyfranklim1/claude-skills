## Glossario de Metricas

### Metricas de Alcance e Visibilidade
- **Impressoes**: Quantas vezes seu anuncio foi exibido (uma pessoa pode ver varias vezes)
- **Alcance**: Quantas PESSOAS diferentes viram seu anuncio
- **Frequencia**: Media de vezes que cada pessoa viu o anuncio (ideal: 1.5 a 3x)
- **CPM (Custo por Mil Impressoes)**: Quanto voce paga para 1.000 pessoas verem seu anuncio

### Metricas de Engajamento
- **CTR (Click-Through Rate)**: % de pessoas que clicaram depois de ver o anuncio
  - Formula: (Cliques / Impressoes) x 100
  - Abaixo de 0.5% = criativo fraco ou publico errado
  - Acima de 2% = excelente
- **CPC (Custo por Clique)**: Quanto voce paga por cada clique
  - Formula: Investimento / Cliques
- **Engajamento**: Curtidas + comentarios + compartilhamentos + salvamentos

### Metricas de Conversao
- **CPA (Custo por Aquisicao)**: Quanto custa cada conversao (lead, venda, cadastro)
  - Formula: Investimento / Conversoes
- **Taxa de Conversao**: % de visitantes que completam a acao desejada
  - Formula: (Conversoes / Cliques) x 100
  - Landing page boa: 3% - 10%
  - Landing page excelente: acima de 10%
- **ROAS (Return on Ad Spend)**: Retorno sobre investimento em anuncios
  - Formula: Receita / Investimento em Ads
  - ROAS 1x = empatou (gastou R$100, faturou R$100)
  - ROAS 2x = dobrou (gastou R$100, faturou R$200)
  - ROAS 3x+ = excelente

### Metricas de Negocio
- **CAC (Custo de Aquisicao de Cliente)**: Custo total para adquirir um cliente
  - Inclui ads + equipe + ferramentas + comissoes
- **LTV (Lifetime Value)**: Quanto um cliente gasta ao longo do tempo
  - Regra de ouro: LTV deve ser pelo menos 3x o CAC
- **Ticket Medio**: Valor medio de cada compra
- **Churn**: Taxa de cancelamento/perda de clientes

## Como Diagnosticar Problemas

### Muitas impressoes, poucos cliques (CTR baixo)
- Criativo nao chama atencao → testar novas imagens/videos
- Copy fraca → reescrever com gatilhos mentais
- Publico errado → revisar segmentacao

### Muitos cliques, poucas conversoes (CPA alto)
- Landing page ruim → melhorar design, copy, velocidade
- Oferta fraca → revisar proposta de valor
- Publico nao qualificado → refinar segmentacao
- Experiencia mobile ruim → testar no celular

### CPC muito alto
- Muita concorrencia no leilao → testar horarios/publicos diferentes
- Relevancia baixa → melhorar criativo e copy
- Publico muito restrito → ampliar segmentacao

### ROAS abaixo de 1x
- Preco do produto muito baixo para o CPA → aumentar ticket ou upsell
- Funil quebrado → mapear cada etapa e encontrar o vazamento
- Fase de aprendizado ainda → esperar 72h antes de mudar

## Framework de Analise Rapida

Para qualquer campanha, avalie nesta ordem:

```
1. ORCAMENTO → Esta gastando o planejado?
2. ALCANCE → Esta alcancando gente suficiente?
3. CTR → As pessoas estao clicando? (problema = criativo)
4. CPC → O clique esta caro? (problema = concorrencia/relevancia)
5. TAXA DE CONVERSAO → Quem clica, converte? (problema = landing page/oferta)
6. CPA → O custo por resultado e sustentavel?
7. ROAS → Esta dando lucro?
```
