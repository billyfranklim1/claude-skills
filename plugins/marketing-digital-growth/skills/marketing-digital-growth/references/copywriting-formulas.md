# Formulas de Copywriting para Anuncios

## Quando usar cada formula

| Formula | Melhor para | Tom |
|---------|-------------|-----|
| AIDA | Anuncios gerais, primeiro contato | Progressivo |
| PAS | Publico com problema claro | Emocional |
| BAB | Transformacao, antes/depois | Aspiracional |
| 4Us | Headlines e titulos curtos | Urgente |
| FAB | Produtos com especificacoes tecnicas | Informativo |
| Storytelling | Reels, videos longos | Narrativo |

## Formula 1: AIDA

**A**tencao → **I**nteresse → **D**esejo → **A**cao

```
ATENCAO: Frase ou visual que para o scroll (hook)
INTERESSE: Informacao relevante que faz a pessoa querer saber mais
DESEJO: Beneficios, prova social, transformacao
ACAO: CTA claro dizendo o que fazer agora
```

**Exemplo - Curso de Confeitaria:**
```
[ATENCAO] Essa tecnica de decoracao fez uma aluna faturar R$ 8.000 no primeiro mes
[INTERESSE] O segredo nao e ter equipamentos caros - e saber as 3 tecnicas que
os confeiteiros profissionais usam e ninguem ensina
[DESEJO] Mais de 2.000 alunas ja transformaram a cozinha de casa em um negocio
lucrativo. Imagine receber encomendas todos os dias...
[ACAO] Clique em "Saiba mais" e garanta sua vaga com 40% de desconto (so ate sexta)
```

## Formula 2: PAS

**P**roblema → **A**gitacao → **S**olucao

A mais poderosa para vendas diretas. Funciona porque ativa a dor antes de apresentar
a cura.

```
PROBLEMA: Descreva o problema que o publico enfrenta (com as palavras dele)
AGITACAO: Amplifique a dor - mostre as consequencias de nao resolver
SOLUCAO: Apresente seu produto/servico como a solucao ideal
```

**Exemplo - Servico de Contabilidade:**
```
[PROBLEMA] Voce paga mais imposto do que deveria porque nao sabe qual regime
tributario e ideal pro seu negocio?
[AGITACAO] Cada mes que passa, dinheiro que poderia estar no seu bolso vai pro
governo. A maioria dos pequenos empresarios perde entre R$ 500 e R$ 3.000 por
mes so por falta de planejamento tributario.
[SOLUCAO] Na [Empresa], fazemos uma analise gratuita do seu negocio e mostramos
exatamente quanto voce pode economizar. Sem compromisso. Clique e agende.
```

## Formula 3: BAB

**B**efore (Antes) → **A**fter (Depois) → **B**ridge (Ponte)

Perfeita para mostrar transformacao.

```
BEFORE: Descreva a situacao atual (com empatia)
AFTER: Pinte o cenario ideal (o que a pessoa quer)
BRIDGE: Mostre como seu produto/servico faz a ponte entre os dois
```

**Exemplo - Academia:**
```
[BEFORE] Cansado de comecar dietas toda segunda-feira e desistir na quarta?
De se olhar no espelho e nao gostar do que ve?
[AFTER] Imagine se sentir confiante, com energia, vestindo as roupas que
quer, e ouvindo "nossa, voce ta diferente!" dos seus amigos.
[BRIDGE] O metodo [Nome] combina treino personalizado + acompanhamento
nutricional + grupo de apoio. 87% dos nossos alunos alcancam seus objetivos
em 90 dias. Comece hoje com uma aula experimental gratis.
```

## Formula 4: 4Us (para Headlines)

**U**rgente → **U**nico → **U**til → **U**ltra-especifico

Use para criar titulos/headlines que chamam atencao.

**Exemplos:**
```
URGENTE: "Ultimas 12 vagas para o workshop de sabado"
UNICO: "O unico metodo de vendas criado por uma mae solo que fatura 6 digitos"
UTIL: "Como reduzir sua conta de luz em 40% sem gastar nada"
ULTRA-ESPECIFICO: "7 modelos de email que geraram R$ 47.832 em 30 dias"
```

## Formula 5: FAB

**F**eature (Caracteristica) → **A**dvantage (Vantagem) → **B**enefit (Beneficio)

Boa para produtos tecnicos ou com diferenciais claros.

```
FEATURE: O que o produto tem/faz (dado tecnico)
ADVANTAGE: Por que isso e melhor que a alternativa
BENEFIT: O que isso significa na vida do cliente
```

**Exemplo - Software de Gestao:**
```
[FEATURE] Dashboard em tempo real com integracao automatica ao seu banco
[ADVANTAGE] Voce nao precisa mais gastar 2 horas por dia lancando dados manualmente
[BENEFIT] Sobra tempo para focar no que realmente importa: crescer seu negocio
```

## Gatilhos Mentais

Use 1-2 gatilhos por copy. Nao force todos de uma vez.

| Gatilho | Como usar | Exemplo |
|---------|-----------|---------|
| Escassez | Limite de vagas/tempo/estoque | "Restam apenas 8 unidades" |
| Urgencia | Prazo definido | "Oferta valida ate meia-noite" |
| Prova social | Numeros, depoimentos | "Mais de 5.000 clientes satisfeitos" |
| Autoridade | Credenciais, midia | "Recomendado pela revista Exame" |
| Reciprocidade | Dar algo gratis antes | "Baixe o guia gratuito" |
| Novidade | Destacar o novo | "Lancamento: metodo inedito de..." |
| Especificidade | Numeros exatos | "Aumente suas vendas em 37%" |
| Antecipacao | Gerar expectativa | "Algo grande esta chegando dia 15..." |

## Estrutura de Anuncio para Meta Ads

### Feed (Instagram/Facebook)
```
HOOK (1a linha): Frase que para o scroll [OBRIGATORIO]
———
CORPO (3-5 linhas): Desenvolva usando formula escolhida
———
CTA: Acao clara + link/botao
```

### Stories/Reels
```
SEGUNDO 1-3: Hook visual + texto curto
SEGUNDO 3-10: Desenvolva o argumento
SEGUNDO 10-15: CTA claro
```

### Carrossel
```
SLIDE 1: Hook (titulo impactante)
SLIDE 2-4: Conteudo de valor (dicas, dados, beneficios)
SLIDE 5: CTA + oferta
```

## Checklist de Copy

Antes de finalizar qualquer copy, verifique:
- [ ] Tem um hook forte na primeira linha?
- [ ] Fala na linguagem do publico (nao em "marketeiro")?
- [ ] Foca em beneficios (nao so caracteristicas)?
- [ ] Tem prova social ou dados?
- [ ] O CTA e claro e especifico?
- [ ] Esta adequado ao formato/plataforma?
- [ ] Gera emocao (curiosidade, urgencia, desejo)?
- [ ] E facil de ler (frases curtas, paragrafos pequenos)?
