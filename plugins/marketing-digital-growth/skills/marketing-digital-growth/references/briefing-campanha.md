# Template de Briefing de Campanha - Meta Ads

Use este template sempre que for montar um plano de campanha para o usuario.

## Estrutura do Briefing

```
========================================
BRIEFING DE CAMPANHA - META ADS
========================================

1. INFORMACOES DO NEGOCIO
   - Nome da empresa/marca:
   - Segmento:
   - Produto/servico anunciado:
   - Diferencial competitivo:
   - URL do site/landing page:

2. OBJETIVO DA CAMPANHA
   - Objetivo principal: [Vendas / Leads / Trafego / Reconhecimento / Engajamento]
   - Meta numerica: [Ex: 100 leads/mes, 50 vendas/mes, 10.000 visitas]
   - Prazo: [30, 60, 90 dias]

3. PUBLICO-ALVO
   - Idade: [faixa]
   - Genero: [M/F/Todos]
   - Localizacao: [cidades, estados, pais]
   - Interesses: [listar 5-10 interesses relevantes]
   - Comportamentos: [compradores online, viajantes, etc.]
   - Dores/problemas do publico:
   - Desejos/aspiracoes:

4. ORCAMENTO
   - Orcamento mensal total: R$
   - Orcamento diario sugerido: R$
   - Distribuicao:
     * Topo de funil (publico frio): XX%
     * Meio de funil (lookalike/morno): XX%
     * Fundo de funil (remarketing): XX%

5. ESTRUTURA DE CAMPANHA
   - Tipo de orcamento: [CBO / ABO]
   - Numero de conjuntos de anuncios:
   - Numero de criativos por conjunto:

   Conjunto 1: [Nome]
   - Tipo de publico: [Interesses / Lookalike / Custom]
   - Segmentacao detalhada:
   - Posicionamentos: [Automatico / Manual - quais]
   - Formato dos criativos: [Imagem / Video / Carrossel]

   Conjunto 2: [Nome]
   - [mesma estrutura]

   Conjunto 3: [Nome]
   - [mesma estrutura]

6. CRIATIVOS
   - Tom de voz: [Profissional / Descontraido / Urgente / Aspiracional]
   - Elementos obrigatorios: [logo, cores da marca, etc.]
   - Formatos necessarios:
     * Feed: 1080x1080 ou 1080x1350
     * Stories/Reels: 1080x1920
   - Quantidade de variacoes: [minimo 3]

7. KPIs E METAS
   - CTR meta: > X%
   - CPC meta: < R$ X
   - CPM referencia: R$ X
   - CPA meta: < R$ X
   - ROAS meta: > Xx

8. CRONOGRAMA
   - Data de inicio:
   - Fase de aprendizado: primeiros 7 dias (nao mexer!)
   - Primeira otimizacao: dia 7-10
   - Revisao completa: dia 14
   - Escala (se resultados positivos): dia 21+

9. PROXIMOS PASSOS
   - [ ] Criar conta no Meta Business Suite
   - [ ] Instalar Pixel do Meta no site
   - [ ] Configurar eventos de conversao
   - [ ] Criar publicos personalizados
   - [ ] Produzir criativos
   - [ ] Subir campanha
   - [ ] Monitorar fase de aprendizado
```

## Dicas para o Briefing

### Para orcamentos pequenos (ate R$ 1.000/mes)
- Foque em 1-2 conjuntos de anuncios apenas
- Use CBO para o Meta otimizar a distribuicao
- Comece com publico de interesses (mais barato testar)
- Minimo de R$ 20/dia para ter dados suficientes

### Para orcamentos medios (R$ 1.000 - R$ 5.000/mes)
- 3-4 conjuntos de anuncios
- Teste CBO vs ABO
- Inclua remarketing
- Comece a testar Lookalike

### Para orcamentos altos (acima de R$ 5.000/mes)
- Estrutura completa com todos os niveis de funil
- Multiplas campanhas (aquisicao + remarketing + retencao)
- Testes A/B sistematicos
- Escala horizontal (novos publicos) e vertical (mais verba)
