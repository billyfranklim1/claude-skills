# Checklist de Erros Fatais por Plataforma

Consulte SEMPRE antes de recomendar algo. Alerte o usuario quando ele estiver
prestes a cometer um destes erros. Cada erro aqui ja custou milhares de reais
a empreendedores.

---

## Erros Fatais Gerais (Todas as Plataformas)

```
🔴 CRITICO — Pode destruir resultados:

□ Nao ter pixel/tag instalado corretamente
  → Voce esta voando as cegas. Instale ANTES de comecar.

□ Nao configurar eventos de conversao
  → O algoritmo nao sabe o que voce quer. Configure compra/lead/cadastro.

□ Lancar campanha sem landing page testada
  → Teste no celular. Teste o formulario. Teste a velocidade. ANTES de gastar.

□ Mexer na campanha nas primeiras 72h (fase de aprendizado)
  → O algoritmo precisa de tempo. Mudancas resetam o aprendizado.

□ Escalar campanha com CPA negativo
  → Se nao ta funcionando em pequeno, NAO vai funcionar em grande.

□ Nao ter UTMs ou cupons para rastrear
  → Sem rastreamento, voce nao sabe qual canal funciona.

□ Copiar estrategia de marca grande sendo negocio pequeno
  → Marcas grandes tem outro orcamento e outro objetivo. Sua realidade e diferente.

□ Todo orcamento em uma unica campanha/criativo
  → Diversifique. Teste. Nao coloque todos os ovos na mesma cesta.
```

## Meta Ads (Facebook/Instagram)

### Erros de Configuracao

```
🔴 Conta pessoal em vez de Business Manager
   → Configure Business Manager. Conta pessoal limita tudo.

🔴 Pixel nao configurado ou configurado errado
   → Verifique: Meta Pixel Helper (extensao Chrome)

🔴 Nao verificar dominio
   → Sem verificacao, eventos perdem dados. Verifique em Business Settings.

🔴 Usar orcamento vitalicio sem saber
   → Iniciante: use orcamento DIARIO. Vitalicio e para experts.

🔴 Selecionar "Otimizar para impressoes" querendo vendas
   → Otimize para CONVERSAO, nao impressoes ou cliques.
```

### Erros de Publico

```
🟡 Publico muito pequeno (< 100K pessoas)
   → Custo sobe e alcance limita. Amplie para 500K-2M.

🟡 Nao excluir quem ja comprou
   → Voce paga para mostrar anuncio pra quem ja e cliente.

🟡 Lookalike sem base boa
   → Lookalike de 50 clientes nao funciona. Precisa de 500+.

🔴 Mudar publico toda semana
   → De tempo ao algoritmo. Minimo 7 dias antes de mudar.
```

### Erros de Criativo

```
🟡 Mais de 20% de texto na imagem
   → Meta nao restringe mais oficialmente, mas ainda entrega menos.

🟡 Mesmo criativo rodando ha mais de 30 dias
   → Fadiga de criativo. Renove a cada 2-3 semanas.

🔴 Nao ter hook nos primeiros 3 segundos (video)
   → Se nao prende em 3s, a pessoa ja scrollou.

🔴 Apenas 1 criativo por conjunto
   → Minimo 3-5 criativos. O algoritmo precisa de opcoes.
```

## Google Ads

### Erros de Configuracao

```
🔴 Nao configurar conversoes no Google Ads
   → Sem conversoes configuradas, Smart Bidding nao funciona.

🔴 Usar correspondencia ampla sem palavras negativas
   → Seu anuncio vai aparecer para buscas totalmente irrelevantes.

🔴 Nao revisar termos de pesquisa semanalmente
   → Voce pode estar pagando por cliques de gente que busca outra coisa.

🔴 Mandar todo trafego para a home do site
   → Cada grupo de anuncios precisa de uma LP especifica.

🔴 Nao usar extensoes de anuncio
   → Extensoes aumentam CTR em 10-15%. Nao usar e desperdicar espaco gratuito.
```

### Erros de Orcamento

```
🟡 Orcamento muito baixo (< R$ 20/dia no Search)
   → Dados insuficientes para otimizar. Melhor focar em menos keywords.

🔴 Definir CPC maximo muito baixo
   → Seu anuncio nao aparece. Comece com CPC sugerido e ajuste.

🔴 Nao definir teto de orcamento para PMax
   → PMax pode gastar muito rapido se nao tiver limites.
```

### Erros de Keywords

```
🔴 Comprar keywords genéricas demais ("sapatos", "advogado")
   → CPC altissimo e conversao baixa. Use long tail.

🟡 Ignorar keywords de marca
   → Concorrentes podem comprar seu nome. Proteja suas keywords de marca.

🔴 Nao agrupar keywords por intencao
   → "como fazer X" e "comprar X" tem intencoes diferentes. Separe.
```

## TikTok Ads

### Erros Especificos

```
🔴 Usar criativo de Instagram no TikTok
   → Formatos diferentes, linguagem diferente. Crie conteudo NATIVO.

🔴 Video muito produzido/profissional
   → No TikTok, "caseiro" performa melhor que "profissional". Pareça nativo.

🔴 Nao ter hook no PRIMEIRO SEGUNDO
   → TikTok tem scroll mais rapido que Instagram. 1 segundo, nao 3.

🟡 Segmentacao muito restrita
   → No TikTok, Broad (amplo) costuma performar melhor.
   → Confie no algoritmo, ele e muito bom em encontrar seu publico.

🔴 Nao renovar criativos semanalmente
   → Fadiga no TikTok e MUITO mais rapida. Criativos duram 7-14 dias.

🟡 Texto longo na caption
   → Ninguem le caption no TikTok. O video e a mensagem.
```

## LinkedIn Ads

### Erros Especificos

```
🔴 Orcamento menor que R$ 2.000/mes
   → CPC do LinkedIn e 5-10x maior que Meta. Orcamento pequeno = poucos dados.

🔴 Usar copy de "vendedor" no primeiro contato
   → LinkedIn funciona com educacao primeiro, venda depois.

🔴 Mandar para landing page generica em vez de Lead Gen Form
   → Lead Gen Forms convertem 2-4x mais no LinkedIn.

🟡 Segmentacao por EMPRESA em vez de CARGO
   → Cargo e mais preciso. "Diretor de Marketing" > "Empresa X"

🟡 Anunciar para publico amplo demais
   → LinkedIn e caro. Cada impressao conta. Segmente com precisao.

🔴 Ignorar o LinkedIn Insight Tag
   → Equivalente ao pixel. Sem ele, sem remarketing, sem dados.
```

## Email Marketing

### Erros Fatais

```
🔴 Comprar lista de emails
   → Ilegal (LGPD), ruins para entregabilidade, queima IP e dominio.

🔴 Nao ter sequencia de boas-vindas
   → O momento de maior engajamento e desperdicado.

🔴 Enviar sem SPF/DKIM/DMARC configurados
   → Seus emails vao direto para spam.

🟡 Mesmo template para todos os segmentos
   → Personalize por segmento. Relevancia = abertura.

🟡 Nao limpar lista de inativos
   → Emails que bounceam destroem sua reputacao de envio.

🔴 Nao ter link de descadastro visivel
   → Alem de ilegal, faz a pessoa marcar como spam.
```

## WhatsApp/DM

### Erros Fatais

```
🔴 Adicionar pessoas em grupos sem permissao
   → Irrita, queima marca, e pode gerar bloqueio do numero.

🔴 Broadcast com mensagem generica
   → Personalize. "Oi amigo" nao funciona. Use o nome.

🔴 Demorar mais de 30 min para responder
   → Cada minuto reduz conversao. Objetivo: < 5 min.

🟡 Mandar audio de 3+ minutos
   → Ninguem ouve. Maximo 60 segundos. Prefira texto.

🔴 Nao ter CRM ou registro de conversas
   → Voce vai esquecer follow-ups e perder vendas.

🟡 Desistir apos 1 tentativa
   → 80% das vendas acontecem entre o 5o e o 12o contato.
```

## SEO

### Erros Fatais

```
🔴 Comprar backlinks
   → Google penaliza severamente. Pode derrubar o site inteiro.

🔴 Conteudo duplicado ou copiado
   → Google nao rankeia e pode penalizar.

🔴 Ignorar mobile
   → Google usa Mobile-First indexing. Site ruim no celular = nao rankeia.

🟡 Focar em volume de posts em vez de qualidade
   → 1 post excelente > 10 posts mediocres.

🔴 Nao configurar Google Search Console
   → E a ferramenta gratuita mais importante do SEO. Configure no dia 1.

🟡 Keywords stuffing (repetir keyword demais)
   → Escreva natural. Google e esperto o suficiente para entender sinonimos.
```
