# Skill: Image Generation (Nano Banana)

## Quando Usar
Quando o usuario pedir para gerar, criar, desenhar, editar imagens ou qualquer conteudo visual.

## Ferramenta
- Gemini CLI com extensao Nano Banana
- Comando base: `gemini --yolo "/generate 'PROMPT_EM_INGLES'"`
- Output: `./nanobanana-output/`

## Comandos Disponiveis
- `/generate 'prompt'` — Gerar imagem
- `/icon 'prompt'` — Gerar icone
- `/diagram 'prompt'` — Gerar diagrama
- `/pattern 'prompt'` — Gerar pattern
- `/restore 'prompt'` — Restaurar imagem
- `/edit 'prompt'` — Editar imagem existente
- `/story 'prompt'` — Gerar story visual

## Regras
- SEMPRE use `--yolo` para nao pedir confirmacao
- SEMPRE escreva o prompt em INGLES para melhor qualidade
- Prompts devem ser detalhados e descritivos
- Para blog: use estilo profissional e moderno
- Para thumbnails: use cores vibrantes e texto grande
